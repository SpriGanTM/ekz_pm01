## Описание проекта

Проект представляет собой веб-сайт для языковой школы, разработанный в рамках экзаменационного задания по модулю **ПМ.01 Проектирование и разработка информационных ресурсов**.

Сайт включает:
- Главную страницу с преимуществами, слайдером отзывов и контактной информацией.
- Страницу «Курсы» со списком актуальных курсов из базы данных и формой записи.
